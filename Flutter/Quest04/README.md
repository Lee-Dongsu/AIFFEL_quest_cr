## 페이지 구현
1. 온보딩 - 앱이 실행되기 이전에 보여지는 화면.
2. 홈 - 메인 화면이자 홈 페이지
3. 투게더 - 운동에 관련된 이벤트를 추천해주는 페이지
4. 피트니스 - 운동에 관련된 각종 컨텐츠를 추천해주는 페이지
5. 내 페이지 - 내 개인 정보 및 운동 기록
